Usage:
    [sudo] ./install.sh [options]

Options:
    -as, --add-shortcuts            Add programs shortcuts to the user's desktop 
                                    directory (do not run as root)
    -ds, --disable-screen-lock      Disable automatic screen lock (work only for
                                    Gnome and KDE)
    -i, -ip, --install-programs     Install programs
    --no-upgrade                    Disables packages upgrades during script run
    --list-programs                 Print programs to install into standard 
                                    output and exit 
    -md, --md                       Enable markdown formatting for outputs
    --debug                         Enable debug info
    -?, -h, --help                  Print help message and exit
